
Simple Calculator Android App

A basic Android application that performs fundamental arithmetic operations such as Addition, Subtraction, Multiplication, and Division using a clean and user-friendly 

interface. Built using Java and XML in Android Studio.

 Features

Two number input fields

Four operations: Add, Subtract, Multiply, Divide

Displays result instantly

Input validation (empty fields & divide-by-zero handling)

Toast messages & AlertDialog confirmation

Simple and elegant UI using CardView & ConstraintLayout

 How to Run

Open Android Studio

Clone this repository

Build and run the project on an emulator or physical device

 Output

User enters two numbers

Selects an operation

Result is displayed on screen
